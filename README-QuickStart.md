# Friska – Quick Start Guide

Welcome to Friska! This guide helps you get started quickly.

---

## 1️⃣ Download

Download the ZIP from GitHub Release:
https://github.com/MuhammadAtta/friska/releases/download/v1.0/friska-v1.0.zip

---

## 2️⃣ Extract

Extract the ZIP anywhere on your computer:
friska/

---

## 3️⃣ Run the Project

### Linux / Mac:
```bash
cd friska
chmod +x setup_friska.sh
./setup_friska.sh
```

### Windows:
Double-click setup_friska.bat or Run as Administrator

---

## 4️⃣ Access
Once the setup completes, open your browser at:

http://localhost:8080

---

## 5️⃣ Notes
Scripts handle dependencies installation, building, and running tests automatically.
Make sure Java 17+ and Maven 3.8+ are installed (for Mac/Linux) or included in PATH (Windows).
For Docker usage, make sure Docker is installed and running if you want to run via Docker.
Enjoy Friska! ⭐
